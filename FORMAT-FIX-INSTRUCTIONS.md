These files are Common Lisp ANSI Spec files, which means the actual content has to stay exactly as it is. 
The files are in MDX format and have custom React Components in them which should not be removed.
You are to fix formatting issues with this files related to:
- New lines are in places where they shouldn't
- Sometimes a new lines are missing
- Code is disperesed without it being surrounded by the code block or inline code format
- The default language for code blocks is `lisp`
Keep in mind that sometimes there are no issues with the text and nothing needs to be changed.

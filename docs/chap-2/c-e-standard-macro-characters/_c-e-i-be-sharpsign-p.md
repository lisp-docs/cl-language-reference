 



#P reads a following *object*, which must be a *string*. 



#P⟨expression⟩ is equivalent to #.(parse-namestring ’⟨expression⟩), except that #P is not affected by **\*read-eval\***. 



For information on how the *Lisp printer* prints <GlossaryTerm styled={true} term={"pathname"}><i>pathnames</i></GlossaryTerm>, see Section 22.1.3.11 (Printing Pathnames).

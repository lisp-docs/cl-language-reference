map-dependents-standard-generic-function
========================================

*Method* **MAP-DEPENDENTS**

**Syntax:**

**map-dependents** (*metaobject* standard-generic-function) *function* => *unspecified*

**Description:**

This is a method on the generic function[map-dependents](/meta-object-protocol/dictionary/map-dependents).

This method has no specified behavior beyond that which is specified for the generic function.

This method cannot be overridden unless the following methods are overridden as well:

-   [add-dependent (*class* standard-generic-function) *dependent*](/meta-object-protocol/dictionary/add-dependent-standard-generic-function)
-   [remove-dependent (*class* standard-generic-function) *dependent*](/meta-object-protocol/dictionary/remove-dependent-standard-generic-function)
